# LA VIE - Next.js Luxury Perfume Store

Stack:
- Next.js
- Tailwind CSS
- Stripe
- Supabase
- PayPal
- Klarna
- Multi-language ready

## Install
npm install

## Run
npm run dev

## Environment
Create `.env.local`

NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
STRIPE_SECRET_KEY=
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=
PAYPAL_CLIENT_ID=

## Features
- Dynamic products
- Real checkout architecture
- Video-ready products
- Music controller
- Auth-ready
- Orders-ready
- CMS-ready
